基本取值

  • int: ${intVal}
    long: ${longVal}
    string: ${stringVal}
    date: ${dateVal?string('yyyy-MM-dd HH:mm:ss')}
    null: ${nullVal!'default value'}
    missing: ${missingVal!'default value'}

运算

  • <#assign x = 100/> x = ${x}
    x + 100 = ${x + 100}

if语句

  • <#assign y = 101/> <#if y == 100> y = 100 <#elseif y == 101> y = 101 <#else> y = 102

if+逻辑运算

  • <#assign xx = 'python'/> <#if xx == 'python' && xx?length==6> xx = python <#elseif xx == 'java' && xx?length==4> xx = java <#else> xx = PHP

switch

  • <#assign yy = 'Java'/> <#switch yy> <#case 'Java'> yy = Java <#break> <#case 'Python'> yy = Python <#break> <#default> yy = PHP
